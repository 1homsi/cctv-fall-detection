## Installation

### Prerequisites

- Python 3.9
- Tensorflow 1.14 (GPU version)
- Keras 2.2.4 (GPU version)
- OpenCV 4.1.0
- Numpy 1.16.4
- Pandas 0.24.2
- Matplotlib 3.1.0
- Scikit-learn 0.21.2
- Scipy 1.3.0
- H5py 2.9.0
- Pydot 1.4.1
- Graphviz 2.38

### Clone

- Clone this repo to your local machine using `git clone`
- Install all the prerequisites
